PROJECT NAME

Pershing307 Member Portal

PROJECT OWNER

General John J. Pershing Lodge No. 307

PROJECT PURPOSE

Build a secure, maintainable, GDPR-conscious member portal for a Masonic Lodge.

The system must be operable by non-technical administrators and remain sustainable for decades.

The portal shall support future deployment for additional lodges without source code modifications.

====================================================

IMPORTANT DEVELOPMENT PRINCIPLES

====================================================

This project shall prioritize:

1. Security
2. Privacy
3. Governance
4. Continuity
5. Maintainability
6. Simplicity
7. Mobile Usability

The project shall NOT prioritize:

- Fancy animations
- Complex visual effects
- Technology experimentation
- Vendor lock-in

SECURITY REQUIREMENTS

====================================================

Mandatory:

- Deny By Default authorization
- Least Privilege
- Protected Domains
- Audit Logging
- Password Hashing
- HTTPS
- Secure Cookies
- SQL Injection Protection
- XSS Protection
- CSRF Protection

Mandatory 2FA:

- PM Super Administrators
- Site Administrators

Recommended 2FA:

- Administrators

====================================================

IMPORTANT DEVELOPMENT PROCESS

====================================================

DO NOT BUILD EVERYTHING AT ONCE.

Implement incrementally.

Start with:

Sprint 1

Authentication
Invitation System
Bootstrap Wizard
Roles
Permissions
Database Foundation

When Sprint 1 is complete:

Stop.

Provide:

- Project structure
- Database schema
- Completed files
- Testing instructions

Wait for review before continuing.

====================================================

SUCCESS CRITERIA

====================================================

A non-technical administrator shall be able to:

- Invite members
- Manage members
- Manage roles
- Change branding
- Change colours
- Change fonts
- Upload banners
- Manage roadmap items

without modifying code.

The portal must remain secure, maintainable and transferable.
